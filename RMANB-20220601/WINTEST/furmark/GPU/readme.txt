	                   ReadMe File of wVGAOC Utility
                           =============================

Overview
========

The wVGAOC.exe is a utility to open VGA overclock mode, and record the VGA clock.


INSTALL
=======

Copy the .zip file to destination directory and extract files and double 
click the icon to run the program. Please pay attentation to the INI's setting.


UNINSTALL
=========

Delete all files from your computer.


Release Package
===============

wVGAOC.EXE              Executable file.
wVGAOC.ini              Generate by wUSBCtl.exe.
HISTORY.TXT             The file is the utility's history.
README.TXT              This is the utility's information.
MSIWmiAcpi.dll          EC control tool.
